Installation:

1. Install Visual Studio 2022 (or later):
https://www.visualstudio.com/en-us/products/visual-studio-community-vs.aspx

2. Install SQL server Developer:
https://www.microsoft.com/en-us/sql-server/sql-server-downloads

3. In SQL Server Management Studio
Right Click the Server -> Properties -> Security, Enable "SQL Server and Windows Authentication mode"

4. Run the SQL script (database.sql) in the project root folder.
